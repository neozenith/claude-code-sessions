`context_ratio(tokens, window)` returns the raw fraction `tokens / window` — e.g. `context_ratio(40_000, 200_000) == 0.2`, `context_ratio(150_000, 200_000) == 0.75` — and `None` when the window is unknown (`context_ratio(50_000, None) is None`). No categorical zone labeling (see G2 ADR: Quantitative ratio only).
